# Introduction to Spring

## What is Spring?

Spring is a Java framework that simplifies Java application development.

Its main responsibility is to create, manage, and provide objects whenever they are required.

---

## Why was Spring created?

As Java applications became larger:

- Managing objects manually became difficult.
- Classes had many dependencies.
- Code became tightly coupled.

Spring solves these problems by managing object creation and dependencies.

---

## Without Spring

The programmer creates objects manually.

```java
Student student = new Student();
```

---

## With Spring

Spring creates and manages objects.

Objects managed by Spring are called **Spring Beans**.

---

## Key Points

- Spring is a Java Framework.
- Spring creates and manages objects.
- Spring reduces boilerplate code.
- Spring helps build large-scale applications.
